TEIWAZIK FORK

SoundCloud Desktop Fork

Нативный клиент SoundCloud на Tauri: без рекламы, с улучшенным плеером, рекомендациями и стабильными автообновлениями.

Скачать последнюю версию Репозиторий форка Основной репозиторий

Что улучшено в форке

  • GO+/HQ streaming и индикация качества потока в интерфейсе.
  • Переработанные рекомендации: Qdrant 96D, hybrid ranking, региональные тренды, LLM rerank.
  • Оптимизация fullscreen/lyrics/visualizer и более плавная работа прогресса.
  • Фиксы аудио-логики (включая crossfade и поведение громкости).
  • Стабильный release-пайплайн и обновлённая подпись updater-артефактов.

Платформы

Windows (exe/msi), Linux (deb/rpm/AppImage/flatpak), macOS (Intel/Apple Silicon).

Технологии

Tauri v2, Rust, React 19, Zustand, TanStack Query, NestJS backend.
